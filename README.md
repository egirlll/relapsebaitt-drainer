# RelapseBaitt Auto-Drain

Auto-drain Chrome extension for [throne.com/relapsebaitt](https://throne.com/relapsebaitt)

## Install
1. Go to `chrome://extensions`
2. Enable "Developer mode"
3. Click "Load unpacked"
4. Select this folder

## How it works
- Opens on throne.com/relapsebaitt
- Auto-detects all items on the page
- Lets you pick which item to drain
- Auto adds to cart → checkout → pay
- Spawns images while draining
